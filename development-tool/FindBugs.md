## 使用

### Bug解释模式
bug可能会引起程序的性能或逻辑问题.
需要说明的是,findbugs能检测的bug pattern远不仅于此,甚至可以定制自己的探测器,因此,这个文档会不断扩充,

| 错误类型                       | 中文解释        |
| -----------------------------: | :-------------- |
| Bad practice                   | 不好的习惯      |
| Correctness                    | 代码的正确性    |
| Dodgy                          | 小问题          |
| Malicious code vulnerability   | 恶意代码        |
| Internationalization           | 国际化问题      |
| Performance                    | 性能问题        |
| Security                       | 安全性问题      |
| Multithreaded currectness      | 线程问题        |
| Experrimental                  | 实验性问题      |
